Qu'est ce que Python ?

  Python est un langage de programmation généraliste. Pour faire simple, il sert à donner des ordres à des logiciels qui vont ensuite executer ces ordres sur tout type d'appareils ou machines.
  Par exemple, nous pouvons dire à une machine de calculer à notre place la moyenne de ces nombres : 51, 29, 37, 24. Celle-ci nous donnera donc le résultat étant 35.25.

Comment fonctionne Python ?

  Python est donc un langage de programmation servant à automatiser des tâches par une machine ou un appareil éléctronique par des ordres. Ceux-ci sont donnés sous forme de mots-clés comme pour chaque langage de programmation. De plus, Python est soumis à une convention (manière d'organiser ses mots-clés dans un ordre qui facilite la vision du code).

Et concrètement... Pourquoi apprendre Python ?

  Beaucoup de gens râlent à propos de Python soit disant trop énervant pour les erreurs que le script (ensemble du code produit) retourne ou pour sa lenteur sur certains projets de développement.
  En réalité ce langage de programmation est un des plus simple (voir le plus simple) qui puisse exister dans son domaine. En effet, si l'on compare deux langages de programmation ensemble puis à Python, on remarque une énorme différence. Pourtant, ils font tout trois les mêmes actions !

  Nous pouvons donc constater par une seule action la facilité que demande Python pour ses programmes. Ici nous avons vu comment afficher quelque chose dans la sortie (shell pour python, console pour JavaScript et C++).
  Cette sortie est en réalité le résultat final de notre script (code/programme) et qui est donc rendu à l'utilisateur.

Un petit récapitulatif

  Python est donc un langage généraliste qui est soumis à une convention. C'est un langage simple d'utilisation fonctionnant comme chaque langage de programmation : avec des mots-clés.
  Python est donc connu pour ces spécifités et vous servira sûrement si vous vous dirigerez vers l'informatique plus tard.

  LEXIQUE :

  • Convention : Modèle d'écriture à suivre lors de la création de programme.
  • Script : Autrement dit "code" ou "programme".
  • Shell : sortie/résultat donné lors du lancement du programme.
