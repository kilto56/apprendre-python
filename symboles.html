Arithmétique (et comparaison)

  Comme vous le savez les symboles peuvent servir dans les mathématiques. Voyons comment fonctionnent-t-il !

L'addition (1/4)

  Aussi simple que cela puisse paraitre, il suffit simplement d'utiliser le symbole "+" pour faire une addition ! Attention car les additions mathématiques ne se font qu'avec les nombres entiers ou flottants !

print(4 + 4) #Donne le nombre 8 de type entier
print(4.5 + 3.5) #Donne le nombre 8.0 de type flottant
print(5.0 + 3) #Donne le nombre 8.0 de type flottant

La soustraction (2/4)

  Aussi simplement que l'addition, il nous faut donc le symbole "-" pour faire une soustraction ! Attention car les soustractions mathématiques ne se font qu'avec les nombres entiers ou flottants !

print(10 - 2) #Donne le nombre 8 de type entier
print(14.5 - 6.5) #Donne le nombre 8.0 de type flottant
print(11.0 - 3) #Donne le nombre 8.0 de type flottant

La multiplication (3/4)

  Avec la multiplication il nous faut le symbole "*" (astérisque) ! Attention car les multiplication mathématiques ne se font qu'avec les nombres entiers ou flottants !

print(4 * 2) #Donne le nombre 8 de type entier
print(3.2 * 2.5) #Donne le nombre 8.0 de type flottant
print(2.0 * 4) #Donne le nombre 8.0 de type flottant

La division (4/4)

  Avec la division nous avons trois signes !

     - "/" Qui permet la division normal.
     - "//" Qui permet la division euclidienne.
     - "%" Qui permet d'avoir le reste de la division euclidienne.
  Attention car les multiplication mathématiques ne se font qu'avec les nombres entiers ou flottants !

print(30 / 4) #Donne le nombre 7.5 de type flottant
print(32 // 4) #Donne le nombre 8 de type entier
print(26.0 % 4) #Donne le nombre 2 de type entier
